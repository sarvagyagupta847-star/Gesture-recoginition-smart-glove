# 🧤 Gesture Recognition smart Glove.

## 📌 Overview
This project is a smart wearable glove that detects hand gestures using the MPU6050 motion sensor and Flex sensors.  
It processes tilt and movement data to recognize different gestures in real-time.

---

## 🚀 Features
- Real-time gesture detection  
- Motion tracking using MPU6050  
- Smooth output using filtering techniques  
- Lightweight and wearable design  
- Easy to expand for IoT (ESP32 / API integration)

---

## 🔧 Components Used
- Arduino Nano  
- MPU6050 (Accelerometer + Gyroscope)  
- Jumper Wires  
- Flex sensors  

---

## ⚙️ How it Works
1. MPU6050 captures motion and tilt data (X, Y, Z axis)
2. Flex sensors detect finger bending positions  
3. Arduino Nano reads sensor values  
4. Data is processed to identify gesture patterns  
5. Output is displayed on Serial Monitor
